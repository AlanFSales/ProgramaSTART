Pre-Processadores CSS: 
	Linguagem intermediária que adicionam  recursos para criar estilos com menos código.
	Interpretador de código.
	Gera CSS a partir de outra sintaxe.
	Ex: LESS,Sass. Stylus.
 - Vantagens: 
	Criação de variaveis, Condicionais, Repetição, Imports, Extends, Funções e Mixins
	Facilita a manutenção do código.
	Permite código limpo.
 - Desvantagens:
	Aumento de complexibilidade.
	Aumenta a chance de baixo desempelho.	

PostCSS: é uma ferramenta que usa plugins baseado em javascript para automatizar operações CSS

Styled Components: é uma biblioteca qua ajudar a escrever código CSS dentro do javascript.

CSS Modules:

Web Components: é uma suíte de diferentes tecnologias que permite criar elementos customizados reutilizáveis.
