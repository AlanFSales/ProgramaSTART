Modos de usar
1 - tag <style> mais indicado usar dentro do <head>
2 - inline. ex: <h1 style="color: red; text-decoration: underline;">
                    Aula de CSS<
                /h1>
3 - arquico .css <link rel="stylesheet" href="style.css" />

} 

seletor por classe:

seletor por id:
