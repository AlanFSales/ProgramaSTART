A java foi criado com o seguinte objetivo: 
Escreva uma vez e execulte em qualquer lugar.
(WORA. White Once, Run Anywhere)

Características:
-Simples
-Orientado a objetos
-Distribuível
-Multithread
-Dinâmico
-Arquitetura neutra
-Portátil
-Alto desempenho
-Robusto
-Seguro

JVM - Máquina Virtual Java

JRE - É o pacote java que possibilita "rodar" programas em java
JDK - É o pacaote java necessário para "desenvolver" programas em java

Literais: representam os dados dos tipos primitivos, dos tipos String e dos tipos nulo.
ex: 	‘a‘      // literal char
	12       // literal int
	true     // literal boolean
	23.4     // litreal double

-Orientação a Objetos(POO): é um paradigma/modelo de programação baseado em classes e objetos.
-Classe: É um elemento da POO que consegue expressar o que temos no mundo real. 
	Ela é formada por variáveis(atributos), contrutor e métodos(Comportamento).
-Método: Determina o comportamento dos objetos.
-Atributo: São elementos que definem a estrutura da classe(variáveis)
-Instanciamento: é criar um objeto(Instanciar uma classe).

4 Pilateres da POO
-Encpsulamento: tem como objetivo evitar que os dados sofram acesso indevido. Isso é feito através de métodos.
-Herança: Uma classe herda caracteristicas de outra classe.
-Polimorfismo: (muitas formas). Um objeto pode se comportar de maneira diferente.
-Abstração: esconder os detalhes, quando usamos um método não precisa saber exatamente o que ele está fazendo.
